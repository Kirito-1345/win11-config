#  Installation Guide
---

## 1. Scoop (Package Manager)

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
```

---

## 2. Git

```powershell
winget install --id Git.Git -e --source winget
```

---

## 3. YASB (Status Bar)

```powershell
winget install --id AmN.yasb
```

> Konfiguration liegt in `yasb widget.txt und yasb styles.txt` — Inhalt in die YASB-Konfigdatei kopieren.

## 6. Yazi (Terminal File Manager)

```powershell
# Yazi selbst
winget install sxyazi.yazi

# Abhängigkeiten
winget install Gyan.FFmpeg 7zip.7zip jqlang.jq oschwartz10612.Poppler sharkdp.fd BurntSushi.ripgrep.MSVC junegunn.fzf ajeetdsouza.zoxide ImageMagick.ImageMagick

# file.exe Pfad setzen (für aktuellen Nutzer)
setx YAZI_FILE_ONE "C:\Program Files\Git\usr\bin\file.exe"

# Für alle Nutzer (erfordert Admin)
setx YAZI_FILE_ONE "C:\Program Files\Git\usr\bin\file.exe" /M
```

## 7. Shell-Erweiterungen

### Autocomplete (PSReadLine)

```powershell
Install-Module -Name PSReadLine -AllowClobber -Force
```

### yt-dlp + FFmpeg

```powershell
winget install yt-dlp
winget install Gyan.FFmpeg
```

### fzf (Fuzzy Finder)

```powershell
winget install -e --id junegunn.fzf
```

### Fastfetch

```powershell
scoop install fastfetch
```

 Terminal & Fastfetch Tutorial: https://www.youtube.com/watch?v=z3NpVq-y6jU
